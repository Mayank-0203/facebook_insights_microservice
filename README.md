# Facebook Insights Microservice

This project is a Facebook Insights Microservice built with FastAPI, providing insights on Facebook pages by scraping data and storing it in a database. The service includes both backend and frontend components.

## Features

- Scrape Facebook page data
- Store data in a database
- Retrieve page details with filtering and pagination
- Simple frontend for user interaction

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/facebook_insights_microservice.git
   cd facebook_insights_microservice
   ```
